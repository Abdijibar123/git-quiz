# Git Quiz



---

1. What is a repository?

asnwer: repository is a container that contains all of your project's files and their history

2. What is the Git command to make a copy of a repo to your laptop?

answer: git clone

3. What is the Git command to send your code to Github?

answer: git push origin main

4. What does the -m in a Git commit command mean or do?

answer: -m stands for message

5. What is the Git command for making a commit?

answer: git commit -m"any massege"

6. What is the Git command to put your files in the staging area?

answer: git add .

7. What does origin stand for when pushing you code?

answer: origin stand for where your repo located or the url of your repo
